# Fiverr Conversation Generator

This project is designed to help you easily generate conversations for Fiverr gigs.

### Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.

### Usage

1. Create a `.env` file from the `.env.example`.
2. Run the server with `node index.js`.
